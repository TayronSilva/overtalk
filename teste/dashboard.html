Guia Rápido

1

Compartilhe o áudio

Clique em "INICIAR SISTEMA" no topo e escolha uma aba com áudio (Discord, YouTube, jogo) para compartilhar. O navegador pede permissão.

2

Fale no microfone

Seu microfone é capturado automaticamente. Use o slider de sensibilidade no painel direito para ajustar. Ative o PTT se preferir falar só quando apertar.

3

Acompanhe as legendas

A tradução aparece em tempo real no feed. As bolhas âmbar são sua voz traduzida, as teal são do áudio do sistema. Você também pode escanear o QR Code com o celular.

4

Exporte o histórico

Use o botão "Exportar Transcrição" para baixar um arquivo da conversa. Ideal para revisar reuniões ou calls depois.
